Publish an app

HTML/CSS/JS or in-browser Python. Apps run sandboxed. Add multiplayer with the built-in Phantom.room() API. Submissions are reviewed before they go live.

Multiplayer & the Phantom API

Every app gets a global Phantom object:

const room = Phantom.room("lobby");       // join a shared room
room.on("move", (data, from) => { ... });  // receive
room.send("move", { x: 10, y: 5 });        // broadcast to everyone else
room.onPresence(state => { ... });         // who's here (live)
room.track({ name: Phantom.me });          // announce yourself
Phantom.me                                  // your @handle

It's backed by realtime websockets, so two players on different devices stay in sync. Python apps reach it via PHANTOM (from Pyodide).
