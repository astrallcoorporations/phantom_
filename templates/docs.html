> ENCRYPTION

Today: TLS in transit, encrypted at rest in your own private database, passwords hashed with Werkzeug (scrypt). Zero trackers, zero telemetry, zero ads.

Roadmap: client-side X25519-XSalsa20-Poly1305 (libsodium) for true E2E DMs — public keys already have a column (profiles.public_key); private keys will never leave the device.

> SELF-HOSTING

Deploy the Flask app anywhere (Vercel config included). Set these environment variables:

envgemini_api_key= # aistudio.google.com/apikey — powers Phantom AI supabase_url= # your database URL supabase_key= # anon key flask_secret_key= # any long random string PUBLIC_URL= # https://your-domain — stabilizes OAuth redirects GOOGLE_CLIENT_ID= GOOGLE_CLIENT_SECRET= GITHUB_CLIENT_ID= GITHUB_CLIENT_SECRET=

OAuth: register PUBLIC_URL + /auth/google/callback (Google Console) and /auth/github/callback (GitHub OAuth app) as the redirect URIs.
